Detailed Description

Feature:

click to switch the page mode to 'facing mode'

Usage:

in template:

<single-page-button @controller="pagemode:SinglePageModeController"></single-page-button>
<dropdown-item @controller="pagemode:SinglePageModeController"></dropdown-item>
<dropdown-button @controller="pagemode:SinglePageModeController"></dropdown-button>
<xbutton @controller="pagemode:SinglePageModeController"></xbutton>

in fragment config:

new PDFUI({
fragments: [{
target: 'single-page',
config: {
callback: UIExtension.controllers.SinglePageModeController
}
}]
});
