# 业务功能快捷键

功能 快捷键 触发时机 addon 说明
复制文本、图片或路径对象 Ctrl+C 选中文本、图片或路径对象 edit-graphics, path-object, text-object 无
删除文本、图片或路径对象 Delete 选中文本、图片或路径对象 edit-graphics, path-object, text-object 无
复制 Annotation Ctrl+C 选中 Annotation 无 无
删除 Annotation Delete 选中 Annotation 无 无
表单宽度增加1px Ctrl+RightArrow 选中表单 form-designer 需要先切换工具到 '选择注释' 或者 '编辑文本、图片或路径'
表单宽度减去1px Ctrl+LeftArrow 选中表单 form-designer 同上
表单宽度增加10px Ctrl+Shift+RightArrow 选中表单 form-designer 同上
表单宽度减去10px Ctrl+Shift+LeftArrow 选中表单 form-designer 同上
表单高度增加1px Ctrl+DownArrow 选中表单 form-designer 同上
表单高度减去1px Ctrl+UpArrow 选中表单 form-designer 同上
表单高度增加10px Ctrl+Shift+DownArrow 选中表单 form-designer 同上
表单高度减去10px Ctrl+Shift+UpArrow 选中表单 form-designer 同上
撤销 Ctrl+Z 无 undo-redo 支持撤销 Annotation 添加、修改以及删除操作
重做 Ctrl+Y 无 undo-redo 支持重做被撤销的 Annotation 添加、修改以及删除操作
展开左侧栏 Ctrl+Right 没有选择 PDF 页面上任何对象 无 无
关闭左侧栏 Ctrl+Left 没有选择 PDF 页面上任何对象 无 无
选择所有 Annotation 和表单 Ctrl+A 切换工具为 '选择注释' 无 无
取消选择 Esc 有选中的 Annotation 或 表单 无 无
切换到手型工具 Esc 没有选中任何 Annotation 或 表单 无 无
快捷搜索 Ctrl+F 无 search 无
显示打印对话框 Ctrl+P 无 print 无
向下滚动可视区域高度 Shift+DownArrow 焦点在PDF文档上 无 如果是单页模式，并且页面高度小于可视区域，则不会向下滚动
向上滚动可视区域高度 Shift+UpArrow 焦点在PDF文档上 无 如果是单页模式，并且页面高度小于可视区域，则不会向上滚动
向下滚动/下一页 DownArrow 焦点在PDF文档上 无 如果使单页模式，并且高度小于可视区域，则会跳到下一页
向右滚动/下一页 RightArrow 焦点在PDF文档上 无 如果使单页模式，则会跳到下一页
向上滚动/上一页 UpArrow 焦点在PDF文档上 无 如果使单页模式，并且高度小于可视区域，则会跳到上一页
向左滚动/上一页 LeftArrow 焦点在PDF文档上 无 如果使单页模式，则会跳到上一页
水平向左滚动可视区域宽度 Shift+LeftArrow 焦点在PDF文档上 无 水平页面模式或者页面宽度大于可视区域
水平向右滚动可视区域宽度 Shift+RightArrow 焦点在PDF文档上 无 水平页面模式或者页面宽度大于可视区域
下一页 PageDown 焦点在PDF文档上 无 单页模式下，页面高度小于可视区域或者页面底部完全显示，则跳到下一页，如果大于可视区域，则滚动可视区域高度或滚动到页面底部
上一页 PageUp 焦点在PDF文档上 无 单页模式下，页面高度小于可视区域或者页面顶部完全显示，则跳到上一页，如果大于可视区域，则滚动可视区域高度或滚动到页面顶部

# 组件快捷键

功能 快捷键 组件 状态
按键触发点击 Enter xbutton,ribbon-button 焦点在按钮上
焦点切换 Tab/Shift+Tab 所有可聚焦组件 可用状态
关闭下拉框 Escape dropdown, ribbon-button 展开下拉列表，且焦点在下拉列表上
关闭弹出层 Escape layer, contextmenu 焦点位于弹出层上
